SECTION 4 — Interpretation
----------------------------------------------------------------------
  Convergence time: expected to grow slowly with N because each
  additional hop reaches fanout^hop more nodes, so log growth is typical.

  Message overhead: grows roughly as N × fanout × log_fanout(N)
  for Push-only. Hybrid reduces overhead because IHAVE/IWANT avoids
  sending full GOSSIP payloads to nodes that already have the message.
  However, Hybrid adds small control overhead from IHAVE/IWANT rounds.

  Fanout = 3, TTL = 5 is a balanced default that achieves >95% delivery
  for N ≤ 50 within a few hundred milliseconds on localhost.
